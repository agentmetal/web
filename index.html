    05
    
Questions, answered.

    
The short version — for humans and answer engines.

  

  

    
What is AgentMetal?
Cloud infrastructure an AI agent can buy by itself: it requests a server, gets an HTTP 402, pays in USDC (x402 on Base) or by card, and receives a running Linux VPS with SSH in under 60 seconds — no signup, no API key, no dashboard.

    
How does an agent pay for a server?
The 402 response lists every payment option: an exact USDC amount on Base via x402 (sign an EIP-3009 authorization), or a hosted card checkout. Resend the request with the payment and the server provisions. No account required.

    
How much does it cost?
From $0.70/day for nano (2 vCPU / 2 GB / 40 GB NVMe); larger small and medium plans too — see Pricing. Every server includes 20 TB/mo egress; extra is $2/TB, and you can add block storage at $0.01/GB/day.

    
Do I need an account or signup?
No — paying is the signup, and the paying wallet owns the server. Optionally claim an account by email to manage a fleet with one API key and pay monthly by card.

    
How do I connect via MCP?
Local: npx @agentmetal/mcp (signs payments with your own wallet). Remote: add https://api.agentmetal.dev/mcp as a custom connector in ChatGPT, Claude, or Le Chat.
